MyFlix

This app will allow users to receive information on movies, directors, and genres so that they can learn more about the movies they are interested in. Users can create a profile to save data about their favorite movies.

Movie API Endpoints


Request URL HTTP Method Request Body Data Response Body Data
Return a list of ALL movies to the user /movies GET None A Json object holding the data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie. Containing a title, description, genre, director and image URL.
{ title: “Fargo”, description: “Movie Description goes here”, genre: “Genre goes here”, director: “Director’s name” imageUrl: }
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/[genre] GET None A JSON object holding data about a genre.
{ genre: “Genres name”, description: “Genres Description” }
Return data about a director (bio, birth year, death year) by name /directors/[name] GET None A JSON object holding data about a director.
{ director: “Director’s Name”, bio: “Bio goes here”, birth: “Birth year goes here”, death: “Death year goes here” }
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{ username: “moviephoneguy”, name: "Cosmo Kramer", dob: “05/01/1943” email: newman@usps.com, password: “jerry5A” } 		A JSON object holding data about the user that was added, including an ID:
{ Id: “unique ID” username: “moviephoneguy”, name: "Cosmo Kramer", dob: “05/01/1943” email: newman@usps.com, password: “jerry5A” }
Allow users to update their user info (username, password, email, date of birth) /user/[username]/[name]/[dob]/[email]/[password] PUT A JSON object holding data about the user to update, structured like:
{ username: “moviephoneguy”, name: "Cosmo Kramer", dob: “05/01/1943” email: newman@usps.com, password: “jerry5A” } 		A JSON object holding data about the user that was updated:
{ username: “moviephoneguy”, name: "Cosmo Kramer", dob: “05/01/1943” email: newman@usps.com, password: “jerry5A” }
Allow users to add a movie to their list of favorites /users/[username]/[favorites]/[title] POST None A text message indicating that the movie was added to favorites.
Allow users to remove a movie from their list of favorites /users/[username]/[favorites]/[title] DELETE None A text message indicating that the movie was removed from favorites.
Allow existing users to deregister /users/[username] DELETE None A message indicating that the user has been removed.
