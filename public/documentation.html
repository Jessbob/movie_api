MyFlix

This app will allow users to receive information on movies, directors, and genres so that they can learn more about the movies they are interested in. Users can create a profile to save data about their favorite movies.

Movie API Endpoints


Request URL HTTP Method Request Body Data Response Body Data
Return a list of ALL movies to the user /movies GET None A Json object holding the data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:Title GET None A JSON object holding data about a single movie. Containing a title, description, genre, director and image URL { Title: “Fargo”, Description: “Movie Description goes here”, Genre: “Genre goes here”, Director: “Director’s name”, ImageUrl: “fargo.png”, Featured: true }
Return data about a genre (description) by name/title (e.g., “Thriller”) movies/genres/:Name GET None A JSON object holding data about a genre.
A JSON object holding data about a genre. { Name: “Genres name”, Description: “Genres Description” }
Return data about a director (bio, birth year, death year) by name /directors/:Name GET None A JSON object holding data about a director.
{ Name: “Director’s Name”, Bio: “Bio goes here”, Birthyear: “Birth year goes here”, Deathyear: “Death year goes here” }
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{ Username: “moviephoneguy”, Birthday: “05/01/1943” Email: newman@usps.com, Password: “jerry5A” } 		A JSON object holding data about the user that was added, including an ID:
{ Id: “unique ID” Username: “moviephoneguy”, Birthday: “05/01/1943” Email: newman@usps.com, Password: “jerry5A” }
Allow users to update their user info (username, password, email, date of birth) /users/:Username PUT A JSON object holding data about the user to update, structured like:
{ Username: “moviephoneguy”, Birthday: “05/01/1943” Email: newman@usps.com, Password: “jerry5A” } 		A JSON object holding data about the user that was updated:
{ Username: “moviephoneguy”, Birthday: “05/01/1943”, Email: newman@usps.com, Password: “jerry5A” }
Allow users to add a movie to their list of favorites /users/:Username/Movies/:MovieID POST None A text message indicating that the movie was added to favorites.
Allow users to remove a movie from their list of favorites /users/:Username/Movies/:MovieID DELETE None A text message indicating that the movie was removed from favorites.
Allow existing users to deregister /users/:Username DELETE None A message indicating that the user has been removed.
